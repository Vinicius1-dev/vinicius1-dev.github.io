# Portfólio de Vinícius Alves

Este site é um portfólio profissional estático em HTML, CSS e JavaScript, preparado para publicação no GitHub Pages.

## Estrutura do projeto

- `index.html` — página principal
- `css/style.css` — estilos do site
- `js/script.js` — interações e comportamento
- `assets/images/` — imagens dos projetos e foto pessoal
- `favicon.svg` — ícone do site
- `robots.txt` — regras básicas para indexação
- `sitemap.xml` — mapa do site para motores de busca

## Publicação no GitHub Pages

1. Crie um repositório no GitHub com o nome `seu-usuario.github.io`.
2. Faça o upload dos arquivos do projeto para a branch `main`.
3. No repositório, acesse `Settings > Pages` e selecione a branch `main` como fonte.
4. Aguarde alguns minutos até o GitHub Pages publicar o site.

## Observações

- O tema escuro/claro é salvo automaticamente no `localStorage`.
- O menu móvel fecha ao selecionar um link.
- O setor de projetos possui filtros funcionais.
- O contato possui botão para copiar o e-mail.
